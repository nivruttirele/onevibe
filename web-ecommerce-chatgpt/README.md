# Web E-commerce ChatGPT Scaffold

This is a React + TypeScript + Vite scaffold project generated from ChatGPT/Figma design workflow.

## Project Overview

This folder contains a basic e-commerce application scaffold with:
- React 18 with TypeScript
- Vite for fast development and building
- Modern project structure

## Workflow

This project was created using the following workflow:
1. Design created in Figma or described to ChatGPT
2. ChatGPT generated the initial project structure and component files
3. Files uploaded to GitHub ecommerce branch for version control and collaboration

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
